# Product Context

This folder stores working product context used to translate natural-language investigation goals into Orbital query, API, catalog, and endpoint-evidence work.

Current files:

- `Cisco_Secure_Client_Endpoint_Context.md`: Windows endpoint context for Cisco Secure Endpoint, Cisco Secure Client, Orbital, and adjacent Cisco endpoint artifacts visible through Orbital/osquery.
- `Orbital_AI_Investigation_Layer_Context.md`: product outcome and workflow context for using Codex skills as a natural-language investigation layer over Cisco Orbital.
- `osquery/`: upstream osquery schema reference files used to validate table and column names for Orbital query work.

Keep this folder method- and product-focused. Do not store live endpoint rows, hostnames, usernames, IP addresses, tenant identifiers, API tokens, raw API responses, or raw investigation output here.
