# Notes And Memory

Store analysis notes, open questions, comparison notes, working assumptions, decision records, and durable query-memory summaries here.

Use this folder for GitHub-synced memory outputs that help future work start from prior conclusions. Store summaries, decisions, and sanitized lessons that improve query quality, speed, and interpretation. Do not store raw endpoint rows, hostnames, usernames, IP addresses, GUIDs, tenant identifiers, credentials, raw API responses, or customer-identifying values.

- `Orbital_Query_Memory.md`: durable record of Orbital queries and API calls completed in Codex chats.
- `decisions/`: decisions that affect query/script behavior, source selection, catalog template reuse, API assumptions, or endpoint safety.
