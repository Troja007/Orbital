# Orbital Query Methods

This folder stores reusable knowledge about how to query endpoint information with Cisco Orbital and osquery.

Store method knowledge only:

- investigation input patterns
- threat or behavior context
- Orbital Catalog `Name` and `ID` references
- useful osquery tables and columns
- SQL patterns
- targeting and platform guidance
- lessons learned

Do not store endpoint query results, hostnames, usernames, IP addresses, GUIDs, customer names, tokens, raw API responses, or raw investigation output.

Use `_template.yaml` for new method records.
